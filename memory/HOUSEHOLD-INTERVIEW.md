# 🧹 บทสัมภาษณ์งานบ้าน — โดย Lisa

> เป้าหมาย: ให้ Lisa รู้จักบ้านและวิถีชีวิตของเจ้านายพอที่จะดูแล stock ของ · วางตารางทำความสะอาด · ทำ checklist · เตือนงานบ้านได้ตรงจริง
> วิธีใช้: ตอบทีละหมวด ไม่ต้องครบทุกข้อ ข้อไหนยังไม่รู้ข้ามได้ Lisa จะเดาที่เหลือจากบริบท

---

## หมวด A — โครงสร้างบ้าน (รู้พื้นที่ก่อน)
1. อยู่บ้านแบบไหน? (คอนโด / บ้านเดี่ยว / ทาวน์เฮาส์ / อพาร์ตเมนต์) กี่ชั้น ขนาดประมาณเท่าไหร่
2. มีห้องอะไรบ้าง? (นอน / น้ำ / ครัว / นั่งเล่น / ทำงาน / ซักล้าง / ระเบียง / เก็บของ)
3. ห้องไหนใช้บ่อยสุด / ห้องไหนรกง่ายสุด
4. มีพื้นที่กลางแจ้งไหม (สวน ระเบียง ที่จอดรถ)

## หมวด B — สมาชิก & คนช่วย
5. อยู่คนเดียวหรืออยู่กับใคร (คน + สัตว์เลี้ยง — โพล่า, ชูว์ครีม)
6. มีแม่บ้าน/คนทำความสะอาดไหม? มาบ่อยแค่ไหน ทำอะไรบ้าง
7. งานบ้านส่วนไหนที่เจ้านายทำเอง vs จ้าง/ฝากคนอื่น

## หมวด C — ความสะอาด (ตั้งตาราง)
8. ปกติทำความสะอาดบ่อยแค่ไหน (รายวัน/สัปดาห์/เมื่อรก)
9. งานที่ทำประจำมีอะไรบ้าง (กวาด ถู ดูดฝุ่น เช็ดโต๊ะ ล้างห้องน้ำ ฯลฯ)
10. งานที่มัก "ลืม" หรือ "ขี้เกียจทำ" คืออะไร — Lisa จะช่วยเตือนเป็นพิเศษ
11. มีมาตรฐานความสะอาดแบบไหน (เนี้ยบมาก / สะอาดพอใช้ได้ / ขอแค่ไม่รก)
12. ช่วงเวลาไหนของวัน/สัปดาห์ที่สะดวกทำงานบ้านที่สุด

## หมวด D — ซักรีด
13. ซักผ้าบ่อยแค่ไหน เครื่องซักเองหรือส่งร้าน
14. มีของที่ต้องแยกซัก/ส่งซักแห้งไหม
15. รีดผ้าเองไหม หรือมีของที่ต้องรีดประจำ (ชุดทำงาน ฯลฯ)

## หมวด E — Stock ของใช้ในบ้าน (หัวใจของ Lisa)
16. ของใช้สิ้นเปลืองที่ต้องเติมประจำมีอะไรบ้าง (กระดาษทิชชู่ น้ำยาล้างจาน ผงซักฟอก สบู่ แชมพู ยาสีฟัน ฯลฯ)
17. ของกินติดบ้านที่ต้องมีตลอด (น้ำดื่ม กาแฟ ของแห้ง เครื่องปรุง)
18. ของโพล่า + ชูว์ครีม ที่ต้องสต็อก (อาหาร ทราย/วัสดุรองกรง ขนม)
19. ปกติซื้อของเข้าบ้านที่ไหน บ่อยแค่ไหน (ตลาด/ห้าง/ออนไลน์ — รายสัปดาห์?)
20. เคยมีของหมดกะทันหันแล้วลำบากไหม อะไรที่ "ห้ามขาด" เด็ดขาด

## หมวด F — ดูแล/ซ่อมบำรุง
21. มีเครื่องใช้ไฟฟ้าที่ต้องดูแลเป็นพิเศษไหม (แอร์ ตู้เย็น เครื่องซัก เครื่องกรองน้ำ)
22. รอบล้างแอร์/เปลี่ยนไส้กรอง/ตรวจเช็กอะไรที่ต้องทำเป็นระยะ
23. มีต้นไม้ในบ้านที่ต้องรดน้ำ/ดูแลไหม (ถ้ามี อาจส่งต่อ Gilbert)

## หมวด G — ความชอบ & ไลฟ์สไตล์
24. กลิ่น/บรรยากาศบ้านที่ชอบ (น้ำหอมปรับอากาศ เทียนหอม โทนสะอาดโล่ง)
25. มีระเบียบเฉพาะตัวไหม (เช่น ห้ามวางของบนโต๊ะ / ของต้องเข้าที่เสมอ)
26. อยากให้ Lisa เตือนแบบไหน (เตือนล่วงหน้า / เตือนตอนเช้า / สรุปรายสัปดาห์)

## 🎯 สิ่งที่ Lisa จะทำได้หลังสัมภาษณ์
- ตั้ง **inventory list** ของที่ต้องสต็อก + เตือนเมื่อใกล้หมด
- วาง **ตารางทำความสะอาด** รายวัน/สัปดาห์ ตามจังหวะชีวิตจริง
- ทำ **checklist บ้าน** ส่งเข้า morning/evening brief ผ่าน Alfred
- เตือน **รอบซ่อมบำรุง** (ล้างแอร์ เปลี่ยนไส้กรอง ฯลฯ)
