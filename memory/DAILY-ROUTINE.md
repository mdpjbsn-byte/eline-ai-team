# Daily Routine — เจ้านาย (Eliné)

**เป้าหมายการนอน:** นอน 21:00 · ตื่น 04:00 (7 ชั่วโมงเต็ม)
**หัวใจ:** เปิดวันด้วยความสงบ (สมาธิก่อนคิด) + ปิดวันด้วยความสงบ (อ่านหนังสือก่อนนอน) — กันไม่ให้ perfectionism คุมทั้งวัน
**หลักการ:** routine ต้องรับใช้เจ้านาย ไม่ใช่เป็นนาย — ไม่ต้องทำเป๊ะทุกข้อ
