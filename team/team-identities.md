## 🎨 โครง 5 บล็อก (Nick) — เต็มตัว + ฉากกว้าง
เรียง: [STYLE] → [SUBJECT] → [ENVIRONMENT] → [LIGHT&COLOR] → [QUALITY]
เปลี่ยนตัวละคร = แก้แค่ SUBJECT + ENVIRONMENT · **ห้ามแตะ STYLE/LIGHT/QUALITY**

**STYLE (เหมือนกันทุกคน):**
```
Black Desert Online game CG style, high-fidelity 3D character render,
semi-realistic stylized game art, Unreal Engine cinematic render,
MMORPG character splash art, hyper-detailed CGI, not a photograph,
```
**QUALITY (เหมือนกันทุกคน):**
```
ultra detailed, 8k, sharp focus, masterpiece, trending on ArtStation,
full character visible, wide angle
```
**Negative:** `photograph, photo, photorealistic, real person, close-up, cropped body, only face, deformed, blurry, low quality, bad anatomy, watermark, text`
**สัดส่วน:** 2:3 (แนวตั้ง เห็นเต็มตัว) · **การจัดเฟรม:** full body shot, dynamic posture, wide environment

> เคล็ดล็อกให้เป็นทีมเดียวกัน: ใช้ Image Guidance → Style Reference รูปแม่ BDO เดียวกัน Strength 0.4–0.6 ทุกคน
